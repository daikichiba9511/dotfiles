# Research Output Formats

Use these formats only when the task benefits from explicit structure.

## Quick Answer

```markdown
## Question
[What was investigated]

## Findings
- [Fact with source]
- [Fact with source]

## Recommendation
- [Best current answer]

## Caveats
- [What is uncertain or constraint-dependent]
```

## Option Comparison

```markdown
## Decision Surface
- Goal: ...
- Constraints: ...

## Options

### Option A
- Strengths:
- Weaknesses:
- Best when:

### Option B
- Strengths:
- Weaknesses:
- Best when:

## Recommendation
- Choose:
- Because:
- Revisit if:
```

## Research Memo

```markdown
## Scope
- Question:
- In scope:
- Out of scope:

## Sources
- [Source 1]
- [Source 2]

## Findings

- ...

### Open questions
- ...

## Trade-offs
- ...

- ...
```
