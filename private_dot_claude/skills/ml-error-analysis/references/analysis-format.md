# ML Error Analysis Format

## Failure Bucket

```markdown
### Bucket
- Name:
- Evidence:
- Frequency / severity:
- Example cases:

### Plausible Causes
- Confirmed:
- Suspected:
- Unknown:

### Next Checks
- ...
```

## Ranked Output

```markdown
## Failure Modes
1. [Highest impact]
2. [Next]

## Hypotheses
- H1:
- H2:

## Recommended Next Experiments
1. [Discriminating test]
2. [Cheap validation]
3. [Higher-cost follow-up]
```
