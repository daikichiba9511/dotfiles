# Global Instructions

## Workflow

- Start in Plan mode for tasks with 3+ steps or architectural decisions. Re-plan if stuck.
- Actively use sub-agents to keep main context clean. Delegate research and parallel analysis.
- Diff against main branch before marking tasks complete.

## File Deletion

- Use `trash` instead of `rm` for file deletion (moves to macOS Trash, recoverable)

## Task Management

- Write plans as checkable items in `tasks/todo.md`. Get confirmation before implementing.
- Check off items as you go. Add a review section on completion.
