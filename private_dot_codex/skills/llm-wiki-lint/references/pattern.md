# LLM Wiki Pattern

Source: Andrej Karpathy's "LLM Wiki" gist, https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f

## Lint Principle

The wiki gains value only if maintenance compounds. Periodic linting should find stale claims, contradictions, missing cross-links, orphan pages, and unanswered questions before the wiki becomes a pile of summaries.
