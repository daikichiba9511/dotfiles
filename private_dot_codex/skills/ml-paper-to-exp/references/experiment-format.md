# ML Paper To Experiment Format

## Extraction Skeleton

```markdown
## Source
- Title:
- URL / citation:
- Problem setting:

## Key Claims
- Claim:
- Evidence:
- Conditions:

## Transferability
- Likely portable:
- Likely source-specific:
- Unknown:
```

## Experiment Card

```markdown
### Experiment N
- Idea:
- Why it might work:
- What must already exist:
- Cheapest valid test:
- Success signal:
- Failure interpretation:
- Cost / risk:
```

## Ranking Summary

```markdown
## Recommended Order
1. [Low-cost validation]
2. [Most promising adaptation]
3. [Higher-risk option]
```
